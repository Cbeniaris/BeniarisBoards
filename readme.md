# Beniaris Boards

Thanks for taking a look at my hand wired keyboards!  Each folder within this repo represents one of the boards I created.  Each keyboard will have an associated readme for information regarding the board itself.  The naming convention is very simple and inspired by @joe-scotto, my last name followed by the number of the keys the board has.  All files are included for each board and you are free to edit and produce these as you wish.
